Setup Guide

Follow these steps before starting Level 1. This is the most important thing you will do. A bad setup breaks everything after it.

Step 1 — Install Termux from F-Droid
⚠ Do NOT use Termux from Play Store. It is old and broken. Use F-Droid only.
  1. Open your phone browser
  2. Go to f-droid.org
  3. Download the F-Droid app and install it
  4. Open F-Droid → search Termux
  5. Install Termux from F-Droid
  6. Open Termux → tap Allow when it asks for storage
Termux — First commands
# First thing after opening Termux
pkg update && pkg upgrade -y
All packages are up to date.
If pkg update fails: Make sure WiFi or mobile data is ON. Try again. If still failing, run: termux-setup-storage first.
Step 2 — Install Debian Inside Termux

Debian is a full Linux system. We run it inside Termux using proot-distro. This gives us access to all the security tools we need.

Installing Debian
# Install proot-distro (the tool that manages Linux systems)
pkg install proot-distro -y
Installed proot-distro.
# Install Debian (takes 5–10 minutes first time)
proot-distro install debian
Downloading Debian rootfs...
Extracting... this takes a while
Installation complete.
# Login to Debian
proot-distro login debian
root@localhost:~#
# You are now inside Debian. Update it.
apt update && apt upgrade -y
All packages upgraded.
If download stops halfway: Run the same command again. It will continue from where it stopped.
Step 3 — Install All Tools for Month 1
⚠ Only install what is listed here. Do NOT install Metasploit or PostgreSQL — they are too heavy for phone and you do not need them in Month 1.
Debian — Tool Installation
# Make sure you are inside Debian first
apt install -y curl wget git nmap whois dnsutils net-tools python3 python3-pip
Done. All tools installed.
# Verify tools are ready
which nmap curl wget git python3
/usr/bin/nmap
/usr/bin/curl
/usr/bin/wget
/usr/bin/git
/usr/bin/python3
Step 4 — Create a Shortcut to Enter Debian

Every time you open Termux, you need to type proot-distro login debian. Make a shortcut so you only type one word.

Termux — Create shortcut
# Exit Debian back to Termux first
exit
# Create a shortcut file
echo 'proot-distro login debian' > ~/deb.sh && chmod +x ~/deb.sh
# Now to enter Debian just type:
bash ~/deb.sh
root@localhost:~#
You are in Debian. Type 'exit' to go back to Termux.
What Termux + Debian Can and Cannot Do
✓ What You CAN Do
Run full Linux commands (nmap, curl, dig)
Write and run bash scripts
Scan networks (with permission only)
Test web applications
Use Git and download tools from GitHub
Run Python scripts and tools
Practice all Month 1 and most Month 2 content
✗ What You CANNOT Do
Run Metasploit (too heavy, needs laptop)
Use WiFi injection adapters (need special hardware)
Run GUI tools like Burp Suite (no desktop)
Run VirtualBox (needs laptop)
Attack real systems without permission (this is illegal)
Run heavy database tools (PostgreSQL will freeze your phone)
Common Problems & Fixes
ProblemFix
pkg update gives errorTurn on WiFi/data. Try again.
Permission denied in TermuxDo not use sudo in base Termux. Just run the command directly.
Not enough spaceFree up 2GB on phone. Delete unused apps and files.
Debian download stops halfwayRun proot-distro install debian again. It continues.
proot-distro: command not foundRun: pkg install proot-distro first
apt command not found in DebianYou are in Termux, not Debian. Run: proot-distro login debian
E: Unable to locate packageRun: apt update first, then try installing again
Phone slows downExit Debian (type exit). Do not run too many tools at once.
Termux session lostOpen Termux again. Run: bash ~/deb.sh to re-enter Debian.
